Variables

Les variables sont des conteneurs pour stocker des données. Elles sont utilisées pour stocker des valeurs pour une utilisation ultérieure.

Définition d'une variable

En programmation, une variable est un conteneur pour stocker des données. Elle est utilisée pour stocker des valeurs pour une utilisation ultérieure dans un programme. Les variables peuvent contenir différents types de données, tels que des nombres, des chaînes de caractères, des listes, des dictionnaires, etc.

Une analogie pour comprendre les variables

Une variable en Python, c’est comme une boîte étiquetée où ranger une valeur pour la retrouver plus tard.

Imaginons que nous ayons une boîte avec une étiquette qui dit “âge”, et à l’intérieur, nous y mettons un papier avec le nombre 25.   • un nom (par exemple age)

  • un type (par exemple int pour un entier).

Nommer une variable

Règles de nomenclature

Les noms de variables doivent respecter des règles de nomenclature imposées par la syntaxe du langage:

  • Le nom d'une variable doit commencer par une lettre (a-z, A-Z) ou un underscore (_).
  • Le nom d'une variable ne peut pas commencer par un chiffre (0-9).
  • Le nom d'une variable ne peut contenir que des caractères alphanumériques et des underscores (a-z, A-Z, 0-9, _).
  • Les noms de variables sont sensibles à la casse (par exemple, age, Age et AGE sont trois variables différentes).

Conventions de nommage

En plus de ces règles, il existe un ensemble de conventions de nommage que les développeurs suivent généralement pour rendre leur code plus lisible et maintenable:

  • Les noms de variables doivent être des noms significatifs qui décrivent la valeur qu'ils stockent.
  • Les noms de variables doivent être en minuscules avec des mots séparés par des underscores (snake_case).
  • Les noms de variables ne doivent pas être des mots-clés réservés par Python.
  • Les noms de variables doivent être courts et concis, mais suffisamment descriptifs pour comprendre leur rôle.

Il est également recommandé d'utiliser l'anglais pour nommer les variables car c'est la langue dominante en programmation, ce qui facilite la compréhension et la collaboration internationale. De plus, la majorité des documentations et ressources sont en anglais, cela assure une consistance avec les bibliothèques et frameworks utilisés.

Prenez ces habitudes dès le début pour écrire un code plus lisible et plus facile à maintenir.

PEP 8

Pour plus de détails sur les conventions de nommage, vous pouvez consulter le PEP 8, qui est le guide de style officiel de Python. PEP 8 est une recommandation pour écrire du code Python lisible et maintenable.

Affectation de valeurs

En Python, une variable est créée en lui affectant une valeur à l'aide de l'opérateur d'affectation =. L'opérateur d'affectation attribue la valeur à droite de l'opérateur à la variable à gauche de l'opérateur.

Vous lirez souvent des phrases comme "déclarer une variable", "initialiser une variable" ou "affecter une variable". Voici leurs significations:

  • Déclarer une variable signifie simplement la créer en lui donnant un nom.
  • Initialiser une variable signifie lui attribuer une valeur pour la première fois.
  • Affecter une variable signifie lui attribuer une nouvelle valeur après son initialisation.

En Python, on combine souvent la déclaration et l'initialisation en une seule étape. Ne vous inquiétez pas trop de ces termes, ils sont souvent utilisés de manière interchangeable.

Afficher le contenu d'une variable

Pour afficher le contenu d'une variable, on utilise la fonction print() qui affiche le contenu de la variable sur la console.

Types de données

En Python, les variables peuvent contenir différents types de données, tels que des nombres, des chaînes de caractères, des listes, des dictionnaires, etc. Contrairement à certains langages de programmation, Python est un langage à typage dynamique, ce qui signifie que le type d'une variable est déterminé automatiquement en fonction de la valeur qui lui est affectée. Cela signifie que vous n'avez pas besoin de spécifier explicitement le type d'une variable lors de sa déclaration.
