## Installation de Python

### Depuis le site officiel

Cette méthode est la plus simple et la plus recommandée si vous débutez.

Rendez-vous sur le site officiel de Python pour télécharger la dernière version de l'interpréteur : https://www.python.org/downloads/

Normalement le site détecte automatiquement votre système d'exploitation et vous propose le bon fichier d'installation.

## Installation d'un IDE (optionnel)

<Callout type="info">
  Si vous savez déjà ce qu'est un IDE et que vous en utilisez un, vous pouvez
  passer cette étape.
</Callout>

Il est possible d'écrire du code dans n'importe quel éditeur de texte, mais l'utilisation d'un IDE (Integrated Development Environment) peut grandement faciliter le développement de logiciels.
Il s'agit d'un logiciel qui fournit des outils complets pour les développeurs afin de faciliter l'écriture, le test et le débogage du code.

Voici quelques avantages à utiliser un IDE :

- **Éditeur de code avancé** : La coloration syntaxique, l'autocomplétion et la gestion des erreurs en temps réel, ce qui rend l'écriture de code plus rapide et moins sujette aux erreurs.
- **Débogage intégré** : Vous pouvez exécuter et déboguer votre code directement dans l'IDE, ce qui vous permet de trouver et corriger les erreurs plus facilement.
- **Contrôle de version** : Un IDE intègre des outils pour travailler avec des systèmes de contrôle de version comme Git, ce qui facilite la gestion des modifications de votre code et la collaboration avec d'autres développeurs.
- **Terminal intégré** : Le terminal intéfré à l'IDE vous permet d'exécuter des commandes sans quitter l'éditeur.

Les deux IDE les plus populaires pour Python sont sans doute **PyCharm** et **Visual Studio Code.**

### Visual Studio Code (préféré)

Visual Studio Code (VSCode) est un éditeur de code open-source et entièrement gratuit développé par Microsoft. Il est léger, rapide et extensible, ce qui en fait un excellent choix pour les développeurs de tous niveaux.
C'est l'IDE que j'utilise au quotidien. Si vous débutez, je vous recommande de commencer par celui-ci.

Pour l'installer, rendez-vous sur le site officiel de Visual Studio Code et téléchargez la version correspondant à votre système d'exploitation: https://code.visualstudio.com/
Installer le logiciel, puis ouvrez-le pour vérifier qu'il fonctionne correctement.

Vous devriez voir une page qui ressemble à celle-ci :

![Visual Studio Code](/vscode.png)

Vous aurez quelques différences liées à la configuration et aux extensions installées, mais l'interface de base devrait être similaire.

### Pycharm

PyCharm est développé par JetBrains, bien connu pour sa suite logiciel de grande qualité.
Il existe une version gratuite de PyCharm, appelée PyCharm Community Edition, qui est suffisante pour la plupart des projets Python.
Pour l'installer, rendez-vous sur le site officiel de JetBrains et téléchargez la version correspondant à votre système d'exploitation: https://www.jetbrains.com/fr-fr/pycharm/

Je n'utilise pas PyCharm personnellement, donc je ne peux pas vous donner de conseils spécifiques pour son utilisation. Cependant, il est très populaire parmi les développeurs Python, donc n'hésitez pas à l'essayer si vous le souhaitez.
